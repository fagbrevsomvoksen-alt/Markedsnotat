Regimeregler

Ved risiko-av: prioriter AVSTÅ eller HOLD – unngå kjøp-forslag.
Standardvalg ved all tvil: AVSTÅ.
Maks én handling per kategori per uke.

Risiko og størrelse (simulert kapital)
  • Foreslått størrelse: 0–1,0 % av hypotetisk kapital
  • Kun lav til moderat risiko aksepteres
  • Alltid konservativ worst-case-kommentar på én linje
Tillatte handlinger per kategori
  • Aksjer: KJØP / SELG / HOLD / AVSTÅ
  • ETF: KJØP / SELG / HOLD / AVSTÅ
  • Indeks: HOLD / AVSTÅ (ikke kjøp eller salg)
  • Krypto: KJØP / SELG / HOLD / AVSTÅ
Sikkerhetsrammer (ufravikelig)
  • Ingen interne data, personopplysninger eller NAV-tilknyttet informasjon
  • Kun generelle, offentlige markedsforhold på overordnet nivå
  • Ikke investeringsråd – ingen oppfordring til kjøp/salg
  • Ingen avkastningsløfter – nøkternt og regelbasert språk
UCITS-krav · Nordnet Europa
  • Alle ETF-instrumenter skal være UCITS-godkjente og tilgjengelige via Nordnet for europeiske investorer
  • US-domisilerte ETFer (VTI, VT, VEA, VWO, BND, GLD) er ikke tillatt – bruk UCITS-ekvivalentene i universet
  • Krypto handles utelukkende via EU-regulerte ETPs (CoinShares XBT Provider) på regulerte børser (Nasdaq Stockholm) – ikke direkte krypto
  • Dersom et instrument ikke er tilgjengelig via Nordnet: skriv «IKKE TILGJENGELIG VIA NORDNET» og velg AVSTÅ
Om Markedsnotat
Regelbasert simulering – ikke tilknyttet reelle markeder eller systemer
VIKTIG: Dette er en ren simulering for lærings- og treningsformål. Ingen reell kapital handles. Ingen personopplysninger benyttes. Ingen tilknytning til NAV, interne systemer eller ansatte. Innholdet er ikke investeringsråd.
Hva er dette?

Et standardisert ukentlig markedsnotat basert på et fast univers av instrumenter. Formålet er å øve på systematisk, regelbasert markedsanalyse. Appen viser notatet for inneværende uke og gir tilgang til universsoversikt og beslutningsregler.

Slik oppdateres appen ukentlig

Åpne index.html i en teksteditor. Finn const NOTE = { ... } øverst i skriptet og oppdater feltene med ukens analyse. Lagre og last siden på nytt. Ingen server eller internett-tilkobling er nødvendig.

Analysemetode

For hvert instrument vurderes: ukentlig trend (positiv / nøytral / negativ), momentum over 4–8 uker, volatilitetsnivå og gjeldende markedsregime. Beslutning fattes kun ved entydige signaler og lav til moderat volatilitet.

UCITS og Nordnet

Bruker er europeisk investor og handler via Nordnet. Alle ETF-instrumenter er UCITS-godkjente og tilgjengelige på Nordnet. US-listede ETFer (VTI, VT osv.) er erstattet med UCITS-ekvivalenter. Krypto-eksponering skjer via CoinShares XBT Provider ETPs på Nasdaq Stockholm – ikke direkte kjøp av kryptovaluta.
